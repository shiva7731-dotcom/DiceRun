# Dice Roll Simulation Game

A simple and interactive Android application built using **Kotlin** and **Jetpack Compose** that simulates rolling a dice. The app generates random dice values and updates the UI dynamically, providing an engaging way to learn state management and UI development in Jetpack Compose.

## 🎲 Project Overview

The Dice Roll Simulation Game is a beginner-friendly Android project that demonstrates how to generate random numbers, update UI components, and manage application state using Jetpack Compose. Users can tap a button to roll a virtual dice and instantly see the result on the screen.

## ✨ Features

* Interactive dice rolling functionality
* Random number generation (1–6)
* Dynamic UI updates using Compose State
* Modern Material Design interface
* Lightweight and user-friendly
* Responsive layout for different screen sizes

## 🚀 Getting Started

### Prerequisites

* Android Studio (Latest Version Recommended)
* Android SDK
* Kotlin Support

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/dice-roll-simulation-game.git
```

2. Open the project in Android Studio.

3. Sync Gradle files.

4. Run the application on an emulator or Android device.
